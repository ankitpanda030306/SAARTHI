### 🏆 5th Runner-Up @ Trithon 2026, Trident Academy of Technology
**Dropout** is an enterprise-grade management engine developed in **24 hours** to address student data fragmentation and attrition tracking. Built by **Team TrojenX**, this project moves beyond simple CRUD operations to provide a stateful, secure, and high-performance environment for academic administrators.

---

## 🚀 Key Features

- **Full-Stack Java Architecture:** Utilizes Vaadin for a server-side "Single Page Application" (SPA) experience, reducing client-side overhead.
- **Enterprise Security:** Integrated **Spring Security** for robust authentication and Role-Based Access Control (RBAC).
- **Persistence Layer:** Advanced Hibernate/JPA mapping for complex relational data handling with **PostgreSQL**.
- **Real-time Dashboard:** A stateful UI that tracks student records (400+) with instant searching, filtering, and sorting.
- **Automated Logging:** Integrated SLF4J logging for system health and connection monitoring.

---

## 🛠️ Tech Stack

- **Backend:** Spring Boot 3.2.4
- **Frontend:** Vaadin Flow (Java-only UI framework)
- **Database:** PostgreSQL (Relational Database Management)
- **Security:** Spring Security (Stateful Session Management)
- **Build Tool:** Maven
